**Problema:** Identificar especies de árboles silvestres requiere conocimientos especializados o acceso a guías impresas, dificultando la educación ambiental y el reconocimiento de la biodiversidad local.

**Solución:** Aplicación web educativa desarrollada con Flask que permite identificar árboles mediante códigos QR. Incluye base de datos científica con información detallada de especies (usos medicinales, maderables, comestibles, etc.), autenticación de usuarios con roles (Administrador/Visitante), generación automática de QR, panel de administración y recuperación de contraseñas vía email.

**Stack:** Python (Flask), MySQL, HTML5, CSS3, JavaScript, Jinja2, qrcode + Pillow, Gunicorn. Despliegue en Railway.app.
